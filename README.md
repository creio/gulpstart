## Установить nodejs
### Windows
- https://nodejs.org/en/

### Установить git
- https://git-scm.com/download/win

### Linux
- curl -sL https://deb.nodesource.com/setup_5.x | sudo -E bash -
- sudo apt-get install -y nodejs
- sudo apt-get install git

### Установка Gulp, Bower глобально
- sudo npm i -g gulp
- sudo npm i -g bower

### В проекте выполнить
- npm i gulp --save-dev
- npm i
- bower i

### Слежка, сервер
- gulp watch или gulp

### Очистка кэша
- gulp clear

### Сборка проекта
- gulp build
